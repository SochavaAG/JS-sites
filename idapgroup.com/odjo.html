All-in DJ console inone SCREEN

ODJO is an iOS app that puts the real size DJ console into one accessible tool for amateur and professional DJs to effortlessly mix and record new tracks.

We had to:

Design intuitive interface matching the real DJ mixer

Develop digital signal processing to preprocess userâ€™s music library

Achieve the scratching of the vinyl without sound stutter

What is ODJO

ODJO is a brilliantly implemented startup idea. The client came up to us bearing in mind the app with a digital DJ toolkit that can be taken anywhere.

Challenges
Real-time audio processing

We wanted users to be able to check out the results of the applied audio effects in real time. We managed to implement that through asynchronous data processing, so user does not have to wait for app to process all the changed and deliver updated result.

Physics-based customizable interface

Client wanted the appâ€™s UI to give off the feeling of the real DJ console. Our designers successfully placed all development on the main screen without making the user switch screens to access multiple audio processing features.

Smart music library

Putting in a vast variety of features for improvisatory DJ mixing, we managed to put in a range of convenient features (Track Preview, Recommended Track, Previously Played track, Queue, Create Playlist, Search) without overloading the app.

Tech applied

We have analyzed the way DJ console works and came up with a pool of technologies that would be the most suitable for transferring its features into the digital environment

ZPLANE

Zplane is what helped us to set up the realistic digital signal processing features, in particular for artifact-free time stretching, pitch shifting, sheet analysis and more. Zplane takes on extraction of high-level musical information such as tempo, rhythm, melody, and key.

Amazing Audio Engine

In the core of the whole real-time music, processing mechanism lays the Amazing Audio Engine framework. It has allowed us to invent the sound reading functionality for scratching from the ground up.

Av Foundation

It is a set of libraries that was used as a base for coding and setting up the Amazing Audio Engine. It is responsible for the app reading and processing unique pieces of sound as well as controlling audiovisual media display.
